Electronic mail security

Email messages are composed, delivered, and stored in a multiple step process, which starts with the message's composition. When the user finishes composing the message and sends it, the message is transformed into a standard format: an RFC 2822 formatted message. Afterwards, the message can be transmitted. Using a network connection, the mail client, referred to as a Mail User Agent (MUA), connects to a Mail Transfer Agent (MTA) operating on the mail server.
